# 🛒 Mini Market Management System

> **Status do Projeto:** 🚧 Em Desenvolvimento 🚧

## 📝 Sobre o Projeto
Este é um sistema web desenvolvido para gerenciar as operações de um mini mercado. O objetivo é facilitar o controle de usuários e produtos utilizando ASP.NET Core MVC.

## ✨ Funcionalidades
- [x] Cadastro de novos usuários.
- [x] Sistema de Sessão (Login/Logout).
- [x] Visualização de Perfil.
- [ ] Cadastro de Produtos.
- [ ] Controle de Estoque.

## 🚀 Tecnologias
* **C#** | **ASP.NET Core MVC**
* **PostgreSQL** (Banco de Dados)
* **Entity Framework Core** (ORM)
* **Bootstrap** (Front-end)
